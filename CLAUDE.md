## Project Summary

An acoustic event classification system using a CNN trained on log-Mel spectrogram
representations. University ML project — Part 1 (planning) is done; Part 2 is active.

**Goal:** Classify short audio clips (≤4 s) into 5 acoustic event classes.
**New in Part 2:** data augmentation, augmentation comparison experiment, online/live system.

## Audio Pipeline — key parameters

```python
# Preprocessing (preprocess.py)
load  → librosa.load(path, sr=SAMPLE_RATE, mono=True)
norm  → audio / max(abs(audio))
trim  → center-crop or zero-pad to int(DURATION * SAMPLE_RATE) = 88200 samples

# Feature extraction (features.py)
mel_S  = librosa.feature.melspectrogram(y=audio, sr=SR, n_fft=N_FFT,
                                         hop_length=HOP_LENGTH, n_mels=N_MELS)
log_S  = np.log(mel_S + 1e-6)          # shape: (128, 173)
tensor = log_S[..., np.newaxis]         # shape: (128, 173, 1)

# Augmentation (training set only — features.py)
aug1 = add_gaussian_noise(audio, std=0.005)
aug2 = time_shift(audio, shift_pct=0.1)
spec_aug = specaugment(log_S, freq_mask=20, time_mask=30)
```

## CNN Architecture (target)

```
Input (128, 173, 1)
→ Conv2D(32, 3×3) + BatchNorm + ReLU + MaxPool(2×2)
→ Conv2D(64, 3×3) + BatchNorm + ReLU + MaxPool(2×2)
→ Conv2D(128, 3×3) + BatchNorm + ReLU
→ GlobalAveragePooling2D
→ Dense(128) + ReLU + Dropout(0.4)
→ Dense(5) + Softmax
```

Loss: `categorical_crossentropy` | Optimizer: `Adam(lr=1e-3)` + `ReduceLROnPlateau`
Primary metric: **macro F1** | Batch: 32 | Max epochs: 50

## Augmentation Experiment

Two full training runs are required:

| Run | Config flag | What changes |
|---|---|---|
| `noaug` | `--augment false` | No augmentation applied |
| `aug` | `--augment true` | Gaussian noise + time shift + SpecAugment |

Both runs saved under `models/noaug/` and `models/aug/`.
Results compared in `results/augmentation_comparison.md`.

## Online System

`src/online.py` — real-time mic capture and inference.

```bash
# Continuous mic mode
python src/online.py --model models/aug/cnn_best.keras

# File fallback (demo without mic)
python src/online.py --model models/aug/cnn_best.keras --file path/to/clip.wav
```

Shared preprocessing: `online.py` imports from `preprocess.py` and `features.py`.
No preprocessing logic may be duplicated in `online.py`.
